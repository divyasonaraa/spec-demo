# Vue3 Composition API + <script setup> + TypeScript. NO Options API.

This project demonstrates the use of Vue 3 Composition API with `<script setup>` syntax and TypeScript. It avoids the Options API for a cleaner and more modern approach.

## Features

- Vue 3 Composition API
- `<script setup>` syntax
- TypeScript support
- Modular and reusable components

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-repo.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:3000`.

## Project Structure

- `src/components`: Contains reusable Vue components.
- `src/types`: TypeScript type definitions.
- `src/views`: Application views.
- `src/assets`: Static assets like images and styles.
